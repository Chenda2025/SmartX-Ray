Setup Guide
Setup Steps
  1. Open Telegram → search @BotFather → send /newbot
  2. Follow the prompts — copy your Bot Token
  3. Message your new bot (any message) — this creates the chat
  4. Click "Discover Chat ID" above (or add it manually to .env)
  5. Click "Send Test Alert" to confirm delivery
Alerts fire automatically — no page needs to be open. The bot sends directly to your Telegram chat 24/7.
.env Variables
TELEGRAM_BOT_TOKEN 123456789:ABC-your-token
TELEGRAM_CHAT_ID your_chat_id
TELEGRAM_ALERT_PNEUMONIA true
TELEGRAM_ALERT_AUTH_FAIL true
TELEGRAM_ALERT_DAILY_SUMMARY false
