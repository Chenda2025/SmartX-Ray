Setup Instructions
  1. Open Telegram → search @BotFather → send /newbot
  2. Follow the prompts and copy your Bot Token
  3. Start your bot, then message @userinfobot to get your Chat ID
  4. Add both values to your .env file: 
    TELEGRAM_BOT_TOKEN=123456789:ABC-your-token-here
TELEGRAM_CHAT_ID=your_chat_id
  5. Restart the server — PNEUMONIA ≥ 80% alerts fire automatically
